# 🏫 Universidad Adventista de Bolivia

## 📘 Taller de Programación

### Laboratorio 3 – PHP (Introducción)

**Objetivo:** Familiarizarse con PHP y su sintaxis básica, practicar la creación y uso de clases para organizar el código.

## Prerrequisitos:
- Tener una cuenta de GitHub
- PHP 8 o superior
- Visual Studio Code

## Descripción
En este laboratorio aprenderás a:
1. Crear un proyecto PHP básico.
2. Definir clases y objetos.
3. Practicar métodos y propiedades de las clases.
